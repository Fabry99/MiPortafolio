Proyectos

Sistema Hotelero

Este sistema fue diseñado para la administración hotelera, ya que permite realizar la reservación de habitaciones, así como gestionar el check-in y el check-out. Además, incluye funcionalidades para la apertura y cierre de caja. El sistema ha sido implementado con dos niveles de acceso: administrativo y recepcionista. Es un sistema escalable, lo que permite su expansión para que otros niveles de empleados puedan utilizarlo en el futuro.
